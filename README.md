# 🔥 FitBlaze — Fitness & Nutrition Tracker

A full-stack fitness and nutrition tracking web application.

## 🚀 Features
- User registration and login (JWT authentication)
- Workout tracking by muscle group
- Daily calorie and nutrition intake monitoring
- Analytics dashboard
- Admin panel for user management
- Nutrition history page

## 🛠️ Tech Stack
**Frontend:** React.js, CSS  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  

## ⚙️ Installation
```bash
# Backend
cd backend
npm install
npm run dev

# Frontend
cd frontend
npm install
npm start
```
